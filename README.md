# Gerenciador de Cursos e Alunos

## Descrição
O projeto tem como objetivo desenvolver uma API REST com FastAPI para gerenciar cursos, turmas, alunos e matrículas no contexto acadêmico. O sistema permite cadastrar cursos com pré-requisitos, criar turmas com controle de vagas e horários, registrar alunos, realizar matrículas com diversas validações e acompanhar o desempenho acadêmico por meio de frequência, notas e relatórios.
